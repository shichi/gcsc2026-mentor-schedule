# GCSC2026 メンター予約システム

## 概要

このディレクトリには、GCSC2026のメンター予約データが保存されています。

## ファイル

- `mentor_bookings.json` - 予約データ（JSON形式）
- `mentor_bookings.md` - 予約状況の可視化（Markdown形式、GitHub上で見やすい）

## 使用方法

### Discord上での予約

チームチャンネル（`#team-01`〜`#team-20`）で以下のコマンドを使用：

```
/mentor book session:session_1 mentor:金子匡俊 & 中島拓成
/mentor cancel session:session_1
/mentor my
/mentor status session:session_1
```

### 予約状況の確認

GitHubで `mentor_bookings.md` を開くと、全体の予約状況を確認できます。
